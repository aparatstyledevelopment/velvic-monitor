# shared/openapi

The contract between backend and frontend.

`spec.json` is generated by FastAPI from the live router definitions:
```
cd backend && python -c "import json; from app.main import create_app; print(json.dumps(create_app().openapi(), indent=2))" > ../shared/openapi/spec.json
```

`types.ts` is generated from `spec.json` by `openapi-typescript`:
```
cd frontend && npm run openapi:gen
```

Generation is run in CI on backend changes; drift between Python types and TS
types is treated as a failed build.
