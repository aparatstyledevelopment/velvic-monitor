# shared/openapi

The contract between backend and frontend.

`spec.json` is generated by FastAPI from the live router definitions; `types.ts`
is then generated from `spec.json` by `openapi-typescript`. Both artifacts are
gitignored — regenerate locally before frontend work that uses them:

```
make openapi
```

That target runs `backend/scripts/dump_openapi.py` (which calls
`create_app().openapi()`) and then `npm run openapi:gen` in `frontend/`. CI
regenerates the artifacts on every run; drift between Python types and TS types
is treated as a failed build.
